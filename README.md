# 🚀 Telecom Data Analysis using Hadoop MapReduce

## 📊 Overview
Analysis of telecom call data using Hadoop MapReduce to extract insights from large-scale datasets.

## 📂 Dataset
Telecom Call Data Records (CDR) dataset in CSV format.

## 📈 Output
- Total Call Duration per User  
- Peak Usage (Day / Evening / Night Calls)  
- Customer Churn Analysis  
- Fraud Detection (High Risk Users)  
- Customer Service Call Analysis  
- Telecom Usage Patterns  

## 🛠️ Technologies
- Hadoop  
- MapReduce  
- Python
